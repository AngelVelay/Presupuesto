# Presupuesto

##Aplicación para la gestion y administración del presupuesto mensual



***Como Empezar*** 

>Clonar el Repo
`git clone https://github.com/AngelVelay/Presupuesto `

> Instalar las dependencias necesarias 
`npm install `

>Ejecuta el comando para ver la app 
`npm start`
